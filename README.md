# 🌐 My Portfolio — Samruddhi Nadgouda

A personal portfolio website built with **HTML**, **CSS**, and **JavaScript**, showcasing my skills, projects, and contact details.

---

## 🔗 Sections

- **Home** — Introduction with a downloadable CV
- **About** — Overview of leadership, web development, volunteering, and community engagement
- **Projects** — Highlights of 4 key projects
- **Contact** — EmailJS-powered contact form with LinkedIn and GitHub links

---

## 🛠️ Featured Projects

| Project | Tech Stack |
|---------|-----------|
| **ZipXtract** — File zipper/unzipper tool | C++, File System Libraries |
| **Talkio** — Real-time chat application | MERN Stack, Socket.IO |
| **ChatConnect** — Multi-client chat server | C++, Winsock2 |
| **Nexus** — Smart contact management system | Spring Boot, MySQL, AWS, OAuth |

---

## ⚙️ Tech Stack

HTML5 · CSS3 · JavaScript · Font Awesome · Google Fonts · EmailJS
